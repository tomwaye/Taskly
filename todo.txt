1️⃣ Core Features (must-haves)

These make Taskly functional as a basic todo app:

    Add tasks

    Input field + “Add” button

    Press Enter to add task

    Display task list

    Show all tasks in a scrollable list

    Task name clearly visible

    Mark tasks as completed

    Checkbox or toggle

    Completed tasks get a strikethrough or faded style

    Delete tasks

    Remove individual tasks

    Edit tasks

    Click on a task to rename it

    Persist tasks

    Save tasks in localStorage so they stay after reload
